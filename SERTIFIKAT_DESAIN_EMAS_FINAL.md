## 🎨 WARNA & GRADIENT

### Border Emas:
```css
background: linear-gradient(135deg, 
    #fbbf24 0%,    /* Kuning terang */
    #f59e0b 50%,   /* Oranye */
    #fbbf24 100%   /* Kuning terang */
);
```

### Nama Emas:
```css
background: linear-gradient(135deg, 
    #fbbf24 0%, 
    #f59e0b 50%, 
    #fbbf24 100%
);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
filter: drop-shadow(0 2px 3px rgba(251, 191, 36, 0.3));
```

### Detail Box (Anggota):
```css
background: linear-gradient(135deg, 
    #f8fafc 0%,    /* Abu-abu sangat muda */
    #e0f2fe 100%   /* Biru sangat muda */
);
border: 2px solid #3b82f6;  /* Biru */
```

### Detail Box (Koperasi):
```css
background: linear-gradient(135deg, 
    #f0fdf4 0%,    /* Hijau sangat muda */
    #d1fae5 100%   /* Hijau muda */
);
border: 2px solid #10b981;  /* Hijau */
```

### Medali Emas:
```css
/* Layer 1 (luar) */
background: linear-gradient(135deg, 
    #fbbf24 0%, 
    #f59e0b 50%, 
    #fbbf24 100%
);

/* Layer 2 (dalam) */
background: linear-gradient(135deg, 
    #f59e0b 0%, 
    #fbbf24 100%
);

box-shadow: 0 6px 20px rgba(251, 191, 36, 0.5);
```

## 🖨️ TIPS CETAK

### Untuk Hasil Terbaik:

#### Kertas:
- **Jenis:** Art Paper 200 gsm (glossy)
- **Ukuran:** A4 (210mm x 297mm)
- **Warna:** Putih glossy untuk efek emas maksimal

#### Printer:
- **Jenis:** Laser warna atau Inkjet photo quality
- **Setting:** Best quality / Photo quality
- **Color:** Full color (CMYK)
- **Orientation:** Portrait
- **Scale:** 100% (no auto-fit)

#### Hasil:
- Border emas akan tercetak dengan gradient indah
- Nama emas akan terlihat mewah
- Medali akan terlihat 3D
