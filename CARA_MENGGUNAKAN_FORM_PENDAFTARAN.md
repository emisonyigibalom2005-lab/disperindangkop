# PANDUAN LENGKAP FORM PENDAFTARAN ANGGOTA KOPERASI

## 🎯 CARA MENGGUNAKAN FORM

### **LANGKAH 1: Buka Halaman Pendaftaran**
```
URL: http://your-domain.com/pendaftaran
```

---

### **LANGKAH 2: Isi Form Step by Step**

#### **📝 STEP 1: DATA PRIBADI**

**Field yang WAJIB diisi (ada tanda bintang merah *):**

1. **NIK** → 16 digit angka sesuai KTP
   ```
   Contoh: 9113221112309001
   ```

2. **Nama Lengkap** → Nama sesuai KTP
   ```
   Contoh: John Doe Kogoya
   ```

3. **Tempat Lahir** → Kota/Kabupaten lahir
   ```
   Contoh: Jayapura
   ```

4. **Tanggal Lahir** → Pilih dari kalender
   ```
   Contoh: 15 Januari 1990
   ```

5. **Jenis Kelamin** → Pilih Laki-laki atau Perempuan

6. **Status Perkawinan** → Pilih: Lajang, Menikah, atau Cerai

7. **Pendidikan Terakhir** → Pilih dari dropdown
   ```
   Pilihan: Tidak Sekolah, SD, SMP, SMA/SMK, D3, S1, S2, S3, Petani
   ```

8. **Agama** → Pilih dari dropdown
   ```
   Pilihan: Kristen, Islam, Katolik, Hindu, Buddha
   ```

9. **No. HP/WhatsApp** → Nomor aktif
   ```
   Contoh: 081234567890
   ```

10. **Email** → Email untuk login (harus unik)
    ```
    Contoh: johndoe@gmail.com
    ```

11. **Password** → Minimal 6 karakter
    ```
    Contoh: password123
    ```

12. **Konfirmasi Password** → Ketik ulang password yang sama

**Setelah semua diisi, klik tombol "Selanjutnya" →**

---

#### **🏠 STEP 2: ALAMAT**

**Field yang WAJIB diisi:**

1. **Distrik** → Pilih distrik tempat tinggal
   ```
   Contoh: Karubaga, Bokondini, Tiom, dll.
   ```

**Field OPSIONAL (boleh dikosongkan):**
- Desa
- Kabupaten (default: Tolikara)
- Alamat Lengkap
- Kode Pos
- Koordinat GPS
- Status Kepemilikan Rumah

**Setelah diisi, klik tombol "Selanjutnya" →**

---

#### **🏪 STEP 3: DATA USAHA**

**Field yang WAJIB diisi:**

1. **Nama Usaha** → Nama usaha Anda
   ```
   Contoh: Toko Sembako Berkah
   ```

2. **Bidang Usaha** → Pilih dari dropdown
   ```
   Pilihan: Pertanian, Perdagangan, Jasa, Industri, Lainnya
   ```

**DATA AHLI WARIS (WAJIB):**

3. **Nama Ahli Waris** → Nama lengkap ahli waris
   ```
   Contoh: Jane Doe Kogoya
   ```

4. **Hubungan Keluarga** → Pilih dari dropdown
   ```
   Pilihan: Suami/Istri, Anak, Orang Tua, Saudara
   ```

5. **No. HP Ahli Waris** → Nomor yang bisa dihubungi
   ```
   Contoh: 081234567891
   ```

6. **NIK Ahli Waris** → 16 digit sesuai KTP
   ```
   Contoh: 9113221112309002
   ```

**Field OPSIONAL:**
- Lama Berdiri Usaha (tahun)
- Jumlah Karyawan
- Modal Usaha (Rp)
- Omzet per Bulan (Rp)
- Alamat Tempat Usaha
- Legalitas Usaha (NIB/SKU/PIRT)
- Keterangan Usaha

**Setelah diisi, klik tombol "Selanjutnya" →**

---

#### **📸 STEP 4: UPLOAD FOTO**

**WAJIB:**

1. **Foto Diri** → Upload foto terbaru
   - Format: JPG, JPEG, atau PNG
   - Maksimal: 2MB
   - Foto akan di-preview sebelum submit

**Tips Foto yang Baik:**
- ✅ Gunakan pencahayaan yang cukup
- ✅ Wajah terlihat jelas dan tidak blur
- ✅ Background bersih dan rapi
- ✅ Foto terbaru (maksimal 6 bulan)

**Setelah foto diupload, klik tombol "Daftar Sekarang" ✓**

## ⚠️ JIKA ADA ERROR

### **Tampilan Error Box:**

Jika ada kesalahan, akan muncul kotak merah besar di atas form seperti ini:

```
╔═══════════════════════════════════════════════════════╗
║  ⚠️  PENDAFTARAN BELUM BISA DIPROSES                 ║
║                                                       ║
║  Terdapat 3 kesalahan pada form yang harus           ║
║  diperbaiki terlebih dahulu.                          ║
║                                                       ║
║  📋 DAFTAR KESALAHAN:                                 ║
║                                                       ║
║  ① NIK harus 16 digit                                 ║
║  ② Email sudah terdaftar. Gunakan email lain         ║
║  ③ Password minimal 6 karakter                        ║
║                                                       ║
║  💡 CARA MEMPERBAIKI:                                 ║
║  • Periksa field dengan border merah                  ║
║  • Data yang sudah diisi tetap tersimpan              ║
║  • Perbaiki field yang error, lalu submit lagi        ║
╚═══════════════════════════════════════════════════════╝
```

### **Cara Memperbaiki Error:**

1. **Baca Error Message** dengan teliti
   - Setiap error diberi nomor (①, ②, ③)
   - Penjelasan jelas apa yang salah

2. **Cari Field yang Error**
   - Field yang error akan ditandai dengan **BORDER MERAH**
   - Ada icon ❌ di sebelah kanan field
   - Ada pesan error di bawah field

3. **Perbaiki Field yang Error**
   - **PENTING:** Data yang sudah diisi TIDAK HILANG
   - Anda hanya perlu memperbaiki field yang error saja
   - Tidak perlu mengisi ulang semua data

4. **Submit Ulang**
   - Setelah diperbaiki, klik "Daftar Sekarang" lagi
   - Form akan divalidasi ulang
   - Jika masih ada error, ulangi langkah 1-3

## 🔍 CONTOH ERROR UMUM DAN SOLUSINYA

### **1. NIK Sudah Terdaftar**
```
❌ Error: NIK yang Anda masukkan sudah terdaftar
```
**Solusi:**
- Cek apakah Anda sudah pernah daftar sebelumnya
- Jika sudah punya akun, gunakan menu Login
- Jika ini kesalahan, hubungi admin

---

### **2. Email Sudah Terdaftar**
```
❌ Error: Email sudah terdaftar. Gunakan email lain atau login jika sudah punya akun
```
**Solusi:**
- Gunakan email yang berbeda
- Atau login jika Anda sudah punya akun

---

### **3. NIK Tidak Valid**
```
❌ Error: NIK harus 16 digit
```
**Solusi:**
- Pastikan NIK 16 digit angka
- Sesuai dengan KTP
- Tidak ada spasi atau karakter lain

---

### **4. Password Tidak Cocok**
```
❌ Error: Konfirmasi password tidak cocok
```
**Solusi:**
- Ketik ulang password dengan benar
- Pastikan password dan konfirmasi password SAMA PERSIS
- Perhatikan huruf besar/kecil

---

### **5. Foto Terlalu Besar**
```
❌ Error: Ukuran foto maksimal 2MB
```
**Solusi:**
- Kompres foto terlebih dahulu
- Gunakan tools online seperti:
  - TinyPNG.com
  - Compressor.io
  - Atau aplikasi edit foto di HP

---

### **6. Format Foto Salah**
```
❌ Error: Foto harus format JPG, JPEG, atau PNG
```
**Solusi:**
- Convert foto ke format JPG/PNG
- Jangan upload file PDF, Word, atau format lain

---

## 📱 TIPS MENGISI FORM

### **✅ DO (Lakukan):**
1. Siapkan data lengkap sebelum mulai mengisi
2. Gunakan NIK dan email yang valid
3. Pastikan nomor HP aktif dan bisa dihubungi
4. Upload foto yang jelas dan terbaru
5. Baca setiap error message dengan teliti
6. Simpan nomor anggota setelah berhasil daftar

### **❌ DON'T (Jangan):**
1. Jangan gunakan NIK atau email orang lain
2. Jangan gunakan password yang mudah ditebak
3. Jangan upload foto yang blur atau gelap
4. Jangan refresh halaman saat proses submit
5. Jangan panik jika ada error - data tidak hilang
6. Jangan submit berkali-kali jika loading

---

## 🆘 TROUBLESHOOTING

### **Problem: Form tidak bisa submit**
**Solusi:**
1. Cek apakah semua field wajib (*) sudah diisi
2. Cek apakah ada error message di atas form
3. Scroll ke atas untuk lihat error box
4. Perbaiki semua error yang ditampilkan

---

### **Problem: Data hilang setelah error**
**Solusi:**
- Ini TIDAK AKAN TERJADI lagi
- Semua data yang sudah diisi akan tetap tersimpan
- Anda hanya perlu perbaiki field yang error

---

### **Problem: Tidak bisa upload foto**
**Solusi:**
1. Cek ukuran foto (maksimal 2MB)
2. Cek format foto (JPG/JPEG/PNG)
3. Coba kompres foto terlebih dahulu
4. Gunakan foto yang berbeda

---

### **Problem: Loading terus tidak selesai**
**Solusi:**
1. Tunggu 1-2 menit
2. Jika masih loading, refresh halaman
3. Cek koneksi internet
4. Coba submit ulang
