## 📍 Cara Akses Menu

### Metode 1: Dari Menu Dokumen Anggota

1. **Login sebagai Admin**
   ```
   URL: /admin/dashboard
   ```

2. **Buka Menu Dokumen Anggota**
   - Klik **"Anggota"** di sidebar
   - Pilih **"Dokumen Anggota"**

3. **Pilih Anggota**
   - Cari anggota yang diinginkan
   - Lihat tombol aksi di kolom terakhir

4. **Klik Tombol Download**
   - **Tombol Biru "Kartu"** 🆔 → Download Kartu Anggota
   - **Tombol Orange "Sertifikat"** 🏆 → Download Sertifikat

### Metode 2: Dari Detail Anggota

1. **Buka Menu Anggota**
   ```
   Admin → Anggota → Daftar Anggota
   ```

2. **Klik Detail Anggota**
   - Pilih anggota
   - Klik tombol **"Detail"** atau **"Lihat"**

3. **Download dari Halaman Detail**
   - Scroll ke bagian bawah
   - Klik tombol **"Download Kartu"** atau **"Download Sertifikat"**

## 🆔 Download Kartu Anggota

### Langkah-langkah:

1. **Akses Menu Dokumen Anggota**
   ```
   Admin → Anggota → Dokumen Anggota
   ```

2. **Cari Anggota**
   - Gunakan filter atau pencarian
   - Temukan anggota yang diinginkan

3. **Klik Tombol "Kartu"**
   - Tombol berwarna **biru** dengan icon 🆔
   - File PDF akan otomatis terdownload

4. **Nama File:**
   ```
   Kartu_Anggota_[Nama_Anggota].pdf
   ```
   Contoh: `Kartu_Anggota_John_Doe.pdf`

### Isi Kartu Anggota:

```
┌─────────────────────────────────────┐
│   KARTU ANGGOTA                     │
│   KOPERASI [NAMA KOPERASI]          │
│   Kabupaten Tolikara                │
├─────────────────────────────────────┤
│  [FOTO]  │ Nama    : [Nama Lengkap] │
│          │ NIK     : [16 digit]     │
│          │ TTL     : [Tempat/Tgl]   │
│          │ Distrik : [Distrik]      │
│          │                          │
│          │ No: [No. Anggota]        │
├─────────────────────────────────────┤
│ Berlaku s/d: [Tanggal + 5 tahun]   │
└─────────────────────────────────────┘
```

### Spesifikasi Kartu:
- **Ukuran:** 85.6mm x 53.98mm (ukuran kartu standar)
- **Orientasi:** Landscape
- **Format:** PDF
- **Background:** Gradient ungu modern
- **Foto:** Otomatis dari database
- **Masa Berlaku:** 5 tahun dari tanggal cetak

## 🖨️ Tips Cetak

### Untuk Kartu Anggota:

#### Bahan yang Direkomendasikan:
1. **Kertas Kartu (Card Stock)**
   - Gramatur: 260-300 gsm
   - Jenis: Art Carton atau Ivory
   - Finishing: Glossy atau Matte

2. **PVC Card**
   - Ukuran: 85.6mm x 53.98mm
   - Ketebalan: 0.76mm (standar)
   - Printer: PVC Card Printer

#### Setting Printer:
```
✅ Paper Size: Custom (85.6mm x 53.98mm)
✅ Orientation: Landscape
✅ Quality: Best/High
✅ Color: Full Color
✅ Margins: Borderless (0mm)
✅ Scale: 100% (No scaling)
```

#### Langkah Cetak:
1. Buka file PDF
2. Klik **Print** (Ctrl+P)
3. Pilih printer
4. Setting sesuai di atas
5. **Print Preview** untuk cek
6. Klik **Print**

#### Finishing:
- ✂️ Potong sesuai ukuran (85.6mm x 53.98mm)
- 📏 Gunakan paper cutter untuk hasil rapi
- 🔒 Laminating untuk ketahanan
- 💳 Bisa tambahkan lanyard/card holder

### Untuk Sertifikat:

#### Bahan yang Direkomendasikan:
1. **Kertas Sertifikat**
   - Gramatur: 100-120 gsm
   - Jenis: Art Paper atau Linen
   - Warna: Putih/Cream

2. **Kertas Fancy**
   - Gramatur: 120-150 gsm
   - Tekstur: Embossed/Linen
   - Warna: Putih/Ivory

#### Setting Printer:
```
✅ Paper Size: A4 (210mm x 297mm)
✅ Orientation: Landscape
✅ Quality: Best/High Quality
✅ Color: Full Color
✅ Margins: Normal (10-15mm)
✅ Scale: Fit to page
```

#### Langkah Cetak:
1. Buka file PDF
2. Klik **Print** (Ctrl+P)
3. Pilih printer
4. Setting **Landscape**
5. Quality: **Best**
6. **Print Preview**
7. Klik **Print**

#### Finishing:
- 🖼️ Bingkai dengan frame A4
- 📜 Bisa digulung dengan pita
- 🎀 Tambahkan seal/stempel koperasi
- ✍️ Tanda tangan manual di tempat yang disediakan

## ❓ FAQ (Frequently Asked Questions)

### Q: Apakah bisa download banyak kartu sekaligus?
**A:** Saat ini download satu per satu. Fitur bulk download bisa ditambahkan jika diperlukan.

### Q: Bagaimana jika foto anggota tidak ada?
**A:** Akan muncul placeholder "No Photo" di kartu. Sebaiknya upload foto terlebih dahulu.

### Q: Apakah bisa edit data setelah download?
**A:** Tidak bisa edit PDF langsung. Harus edit data anggota di sistem, lalu download ulang.

### Q: Masa berlaku kartu berapa lama?
**A:** Otomatis 5 tahun dari tanggal cetak. Bisa diubah di template jika perlu.

### Q: Apakah bisa cetak di printer biasa?
**A:** Bisa! Untuk kartu, cetak di kertas A4 lalu potong. Untuk sertifikat, langsung cetak A4 landscape.

### Q: Bagaimana cara menambahkan logo koperasi?
**A:** Upload logo di setting koperasi, atau edit template kartu-sertifikat.blade.php

### Q: Apakah bisa export ke format lain (JPG, PNG)?
**A:** PDF bisa dikonversi ke JPG/PNG menggunakan tools online atau Adobe Acrobat.
