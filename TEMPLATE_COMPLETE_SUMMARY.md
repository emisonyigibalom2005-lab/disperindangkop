## 🖨️ TIPS CETAK

### Kartu (85.6mm x 53.98mm):
- **Bahan:** PVC atau kartu plastik
- **Finishing:** Laminasi untuk ketahanan
- **Tempat:** Print shop khusus kartu
- **Catatan:** Ukuran standar kartu kredit/ATM

### Sertifikat (A4 Portrait):
- **Bahan:** Art Paper 150-200 gsm
- **Cetak:** Full color (CMYK)
- **Finishing:** Laminasi atau frame untuk display
- **Catatan:** Pastikan printer mendukung A4

### Dokumen Word:
- **Bahan:** HVS 80 gsm atau A4 biasa
- **Cetak:** Hitam putih atau warna
- **Finishing:** Staples atau dijilid
- **Catatan:** Bisa diedit sebelum cetak

## 🔧 TROUBLESHOOTING

### Jika Logo Tidak Muncul:
```bash
# Clear cache
php artisan view:clear
php artisan cache:clear

# Restart server
php artisan serve
```

### Jika PDF Tidak Ter-generate:
```bash
# Pastikan package DomPDF terinstall
composer require barryvdh/laravel-dompdf

# Clear cache
php artisan config:clear
```

### Jika Dokumen Word Tidak Download:
- Pastikan browser tidak memblokir download
- Cek permission folder storage
- Clear browser cache
