# 🚀 QUICK GUIDE - Pengaturan Sistem

## 📖 Panduan Cepat Menggunakan Fitur Pengaturan Sistem

### 🎯 Apa itu Pengaturan Sistem?

Fitur yang memungkinkan Admin mengubah **logo, warna, dan informasi** sistem tanpa perlu edit kode. Semua perubahan **otomatis diterapkan** ke seluruh sistem!

---

## 🔐 Cara Mengakses

1. Login sebagai **Admin**
2. Lihat sidebar kiri
3. Klik menu **"Pengaturan Sistem"** (icon ⚙️)
4. Atau akses langsung: `http://your-domain.com/admin/settings`

---

## 📑 3 Tab Pengaturan

### 1️⃣ Tab UMUM
Mengatur informasi dasar aplikasi

| Field | Deskripsi | Contoh |
|-------|-----------|--------|
| **Nama Aplikasi** | Nama lengkap sistem | DISPERINDAGKOP |
| **Nama Singkat** | Nama pendek/subtitle | Kab. Tolikara |

**Tampil di:**
- ✅ Title browser (tab)
- ✅ Navbar
- ✅ Footer
- ✅ Halaman login

---

### 2️⃣ Tab TAMPILAN
Mengatur logo dan warna sistem

#### 📷 Logo & Icon

| Field | Deskripsi | Format | Max Size |
|-------|-----------|--------|----------|
| **Logo Utama** | Logo di sidebar & navbar | JPG, PNG, SVG | 2 MB |
| **Logo Login** | Logo di halaman login | JPG, PNG, SVG | 2 MB |
| **Favicon** | Icon di tab browser | PNG, ICO | 2 MB |

**Tips Upload Logo:**
- Gunakan background transparan (PNG)
- Ukuran rekomendasi: 200x200px
- File kecil agar loading cepat

#### 🎨 Warna Tema

| Warna | Fungsi | Default | Digunakan Untuk |
|-------|--------|---------|-----------------|
| **Primary** | Warna utama | #1a3a6e (Biru) | Navbar, sidebar, button utama |
| **Secondary** | Warna kedua | #3b82f6 (Biru muda) | Gradient, hover effect |
| **Sidebar** | Warna sidebar | #1a3a6e (Biru) | Background sidebar admin |
| **Success** | Warna sukses | #10b981 (Hijau) | Notifikasi berhasil, status aktif |
| **Warning** | Warna peringatan | #f59e0b (Kuning) | Alert, tombol login |
| **Danger** | Warna bahaya | #ef4444 (Merah) | Error, tombol hapus |

**Tips Memilih Warna:**
- Gunakan warna yang kontras agar mudah dibaca
- Primary & Secondary sebaiknya satu keluarga warna
- Test di mode terang dan gelap

**Tampil di:**
- ✅ Admin panel
- ✅ Halaman login
- ✅ Website public
- ✅ Semua button dan UI element

## 💾 Cara Menyimpan Perubahan

### Simpan Perubahan:
1. Ubah field yang diinginkan
2. Klik tombol **"💾 Simpan Perubahan"** (biru)
3. Tunggu notifikasi sukses
4. **Refresh halaman** untuk melihat perubahan

### Reset ke Default:
1. Klik tombol **"🔄 Reset ke Default"** (merah)
2. Konfirmasi dengan klik "Ya, Reset!"
3. Semua setting kembali ke nilai awal
4. Refresh halaman

## 🎯 Contoh Penggunaan

### Skenario 1: Ganti Logo
**Tujuan**: Mengganti logo lama dengan logo baru

**Langkah:**
1. Buka Tab **Tampilan**
2. Klik **"Pilih File"** di Logo Utama
3. Pilih file logo baru (PNG/JPG)
4. Preview akan muncul otomatis
5. Klik **"Simpan Perubahan"**
6. Refresh halaman
7. ✅ Logo baru tampil di sidebar, navbar, dan login

---

### Skenario 2: Ubah Warna Tema
**Tujuan**: Mengganti warna biru menjadi hijau

**Langkah:**
1. Buka Tab **Tampilan**
2. Klik color picker **Warna Primary**
3. Pilih warna hijau (contoh: #10b981)
4. Klik color picker **Warna Secondary**
5. Pilih hijau muda (contoh: #34d399)
6. Klik **"Simpan Perubahan"**
7. Refresh halaman
8. ✅ Semua elemen berubah jadi hijau

---

### Skenario 3: Update Kontak
**Tujuan**: Mengganti nomor telepon dan email

**Langkah:**
1. Buka Tab **Kontak**
2. Ubah **Email** menjadi: `admin@example.com`
3. Ubah **Telepon** menjadi: `0812345678`
4. Klik **"Simpan Perubahan"**
5. Refresh halaman
6. ✅ Kontak baru tampil di navbar dan footer

## ❓ FAQ (Frequently Asked Questions)

### Q: Perubahan tidak tampil setelah save?
**A:** Refresh halaman dengan `Ctrl + F5` (hard refresh) atau clear browser cache.

### Q: Upload logo gagal?
**A:** Pastikan:
- File format: JPG, PNG, atau SVG
- Ukuran file: Maksimal 2 MB
- File tidak corrupt

### Q: Warna tidak berubah?
**A:** 
1. Pastikan sudah klik "Simpan Perubahan"
2. Refresh halaman dengan `Ctrl + F5`
3. Clear browser cache jika masih tidak berubah

### Q: Bisa reset ke default?
**A:** Ya! Klik tombol "Reset ke Default" di bagian bawah halaman.

### Q: Perubahan berlaku di mana saja?
**A:** Perubahan berlaku di:
- ✅ Admin Panel (semua role)
- ✅ Halaman Login
- ✅ Website Public/User
- ✅ Navbar & Footer
- ✅ Semua halaman sistem

### Q: Apakah perlu edit kode?
**A:** **TIDAK!** Semua bisa diubah dari halaman Pengaturan Sistem.

### Q: Bisa undo perubahan?
**A:** Tidak ada fitur undo, tapi bisa:
1. Catat nilai lama sebelum ubah
2. Atau gunakan "Reset ke Default"

## ⚠️ Tips & Peringatan

### ✅ DO (Lakukan):
- Backup logo lama sebelum ganti
- Test warna di berbagai halaman
- Gunakan logo dengan background transparan
- Pilih warna yang kontras dan mudah dibaca
- Refresh halaman setelah save

### ❌ DON'T (Jangan):
- Upload file terlalu besar (>2MB)
- Gunakan warna terlalu terang/gelap
- Lupa klik "Simpan Perubahan"
- Panik jika ada error (bisa reset ke default)
