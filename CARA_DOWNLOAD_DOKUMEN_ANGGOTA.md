# 📄 Cara Download Dokumen Anggota

## Fitur Dokumen Anggota

Sistem ini menyediakan fitur lengkap untuk mengelola dan mengunduh dokumen anggota koperasi dalam berbagai format.

## 🎯 Fitur yang Tersedia

### 1. **Dokumen Anggota (Word)**
Dokumen lengkap berisi semua data anggota dalam format Microsoft Word (.doc)

**Isi Dokumen:**
- ✅ Data Pribadi (NIK, Nama, TTL, Jenis Kelamin, Agama, dll)
- ✅ Informasi Kontak (No. HP, Email)
- ✅ Alamat Lengkap (Desa, Distrik, Kabupaten, Koordinat GPS)
- ✅ Data Usaha (Nama Usaha, Bidang, Modal, Omzet)
- ✅ Data Keuangan (Simpanan, Rekening Bank, NPWP)
- ✅ Data Ahli Waris
- ✅ Status Keanggotaan
- ✅ Foto Anggota (jika ada)

### 2. **Kartu Anggota (PDF)**
Kartu identitas anggota koperasi dalam format PDF siap cetak

**Spesifikasi:**
- 📏 Ukuran: 85.6mm x 53.98mm (ukuran kartu standar)
- 🎨 Design: Modern dengan gradient background
- 📸 Foto anggota
- 🔢 Nomor anggota
- ℹ️ Data pribadi ringkas
- ⏰ Masa berlaku 5 tahun

### 3. **Sertifikat Keanggotaan (PDF)**
Sertifikat resmi keanggotaan koperasi dalam format PDF

**Spesifikasi:**
- 📄 Ukuran: A4 Landscape
- 🎨 Design: Formal dengan border elegant
- 🏆 Logo koperasi
- 📝 Nama anggota dan nomor anggota
- ✍️ Tempat tanda tangan Ketua dan Sekretaris
- 🗓️ Tanggal penerbitan

## 📍 Cara Mengakses

### Langkah 1: Masuk ke Menu Dokumen Anggota
1. Login sebagai **Admin**
2. Klik menu **"Anggota"** di sidebar
3. Pilih submenu **"Dokumen Anggota"**

### Langkah 2: Lihat Daftar Anggota
Anda akan melihat:
- 📊 **Statistik Dokumen:**
  - Total Anggota
  - Dokumen Lengkap
  - Dokumen Sebagian
  - Dokumen Tidak Lengkap

- 📋 **Tabel Anggota** dengan kolom:
  - Nama Anggota
  - No. Anggota
  - Koperasi
  - No. HP
  - Status Dokumen (KTP, KK, Foto)
  - Tombol Aksi

### Langkah 3: Filter & Cari (Opsional)
**Filter berdasarkan:**
- Status Dokumen (Lengkap/Sebagian/Tidak Lengkap)
- Pencarian (Nama atau No. Anggota)

**Cara Filter:**
1. Pilih status dokumen dari dropdown
2. Ketik nama/no anggota di kolom pencarian
3. Klik tombol **"Cari"**
4. Klik **"Reset"** untuk menghapus filter

## 📥 Cara Download Dokumen

### Download Dokumen Word
1. Cari anggota yang ingin diunduh
2. Klik tombol **"Word"** (hijau) 📗
3. File akan otomatis terdownload dengan nama:
   ```
   Dokumen_Anggota_[Nama]_[No_Anggota].doc
   ```

### Download Kartu Anggota
1. Cari anggota yang ingin diunduh
2. Klik tombol **"Kartu"** (biru) 🆔
3. File PDF akan otomatis terdownload dengan nama:
   ```
   Kartu_Anggota_[Nama].pdf
   ```
4. Cetak kartu menggunakan printer kartu atau kertas A4

### Download Sertifikat
1. Cari anggota yang ingin diunduh
2. Klik tombol **"Sertifikat"** (orange) 🏆
3. File PDF akan otomatis terdownload dengan nama:
   ```
   Sertifikat_[Nama].pdf
   ```
4. Cetak sertifikat di kertas A4 landscape

## 💡 Tips & Trik

### Untuk Cetak Kartu Anggota:
1. ✅ Gunakan kertas kartu (260-300 gsm)
2. ✅ Setting printer: Borderless printing
3. ✅ Potong sesuai ukuran (85.6mm x 53.98mm)
4. ✅ Laminating untuk ketahanan

### Untuk Cetak Sertifikat:
1. ✅ Gunakan kertas A4 berkualitas (100-120 gsm)
2. ✅ Setting: Landscape orientation
3. ✅ Cetak dengan kualitas tinggi (Best/High Quality)
4. ✅ Bingkai untuk tampilan lebih formal

### Untuk Dokumen Word:
1. ✅ Bisa diedit setelah download jika perlu
2. ✅ Simpan sebagai backup data anggota
3. ✅ Bisa dikonversi ke PDF jika diperlukan
4. ✅ Cocok untuk arsip digital

## ❓ Troubleshooting

### Dokumen tidak terdownload?
1. Cek koneksi internet
2. Pastikan browser mengizinkan download
3. Coba browser lain (Chrome/Firefox)
4. Clear cache browser

### Foto tidak muncul di dokumen?
1. Pastikan anggota sudah upload foto
2. Cek path foto di storage
3. Pastikan file foto masih ada

### Kartu/Sertifikat tidak bisa dicetak?
1. Pastikan PDF reader terinstall
2. Cek setting printer
3. Pastikan kertas sesuai ukuran
4. Coba print preview dulu
