## 🖨️ Cara Menggunakan

### Langkah 1: Akses Sertifikat
1. Login sebagai **Admin**
2. Buka menu **Laporan** → **Sertifikat Koperasi**
3. Klik tombol **"Cetak PDF"** (kuning) pada koperasi yang diinginkan

### Langkah 2: Pilih Metode Print

#### Metode A: Print Biasa
1. Klik tombol **"Print"** (hijau)
2. Atau tekan **Ctrl+P**
3. Dialog print akan muncul
4. Atur pengaturan print
5. Klik Print

#### Metode B: Auto Print (REKOMENDASI!)
1. Klik tombol **"Auto Print"** (biru)
2. Dialog print langsung muncul otomatis
3. Atur pengaturan print
4. Klik Print

#### Metode C: Keyboard Shortcut
1. Tekan **Ctrl+P** (Windows) atau **Cmd+P** (Mac)
2. Dialog print langsung muncul
3. Atur pengaturan print
4. Klik Print

### Langkah 3: Pengaturan Print
- **Orientasi**: Landscape (WAJIB!)
- **Ukuran**: A4
- **Margin**: None atau Minimum
- **Background Graphics**: Enabled (WAJIB!)
- **Kualitas**: High/Best

### Langkah 4: Print atau Save
- Klik **Print** untuk langsung cetak
- Atau pilih **Save as PDF** untuk simpan digital

## 💡 Tips & Trik

### Untuk Hasil Print Terbaik:
1. **Kertas**: Gunakan HVS 100 gsm atau kertas sertifikat
2. **Printer**: Printer color untuk hasil optimal
3. **Kualitas**: Set ke High atau Best quality
4. **Background**: WAJIB diaktifkan
5. **Orientasi**: WAJIB Landscape
6. **Preview**: Gunakan Print Preview sebelum print

### Untuk Arsip Digital:
1. Klik Print atau Auto Print
2. Pilih **"Save as PDF"** sebagai printer
3. Klik Save
4. PDF siap untuk disimpan atau dikirim via email

### Untuk Print Batch:
1. Buka sertifikat pertama
2. Klik Auto Print
3. Print atau Save
4. Klik Kembali
5. Ulangi untuk sertifikat berikutnya

## 🔧 Troubleshooting

### Logo tidak muncul?
**Solusi**:
- Pastikan file `public/logo.png` ada
- Cek permission folder public
- Refresh browser dengan Ctrl+F5

### Background tidak tercetak?
**Solusi**:
- Aktifkan "Background Graphics" di print settings
- Atau "Print Background Colors and Images"

### Layout tidak pas?
**Solusi**:
- Pastikan orientasi **Landscape**
- Set margin ke **None** atau **Minimum**
- Scale harus **100%**

### Tombol tidak berfungsi?
**Solusi**:
- Pastikan JavaScript enabled
- Refresh halaman
- Clear browser cache
