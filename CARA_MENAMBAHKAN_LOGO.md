# Cara Menambahkan Logo pada Laporan

## Logo yang Dibutuhkan

Untuk tampilan laporan yang rapi dan profesional, sistem membutuhkan 2 logo:

1. **Logo Kabupaten Tolikara** - Sudah tersedia di `public/images/logo-tolikara.png`
2. **Logo Koperasi** - Perlu ditambahkan di `public/images/logo-koperasi.png`

## Cara Menambahkan Logo Koperasi

1. Siapkan file logo koperasi dalam format PNG dengan latar belakang transparan
2. Ukuran yang disarankan: 500x500 pixel atau lebih (rasio 1:1)
3. Upload file logo ke folder `public/images/` dengan nama `logo-koperasi.png`
4. Pastikan file dapat diakses melalui browser di URL: `http://your-domain/images/logo-koperasi.png`

## Spesifikasi Logo

- **Format**: PNG (dengan transparansi)
- **Ukuran**: Minimal 500x500 pixel
- **Rasio**: 1:1 (persegi)
- **Kualitas**: High resolution untuk hasil print yang baik

## Lokasi Penggunaan Logo

Logo akan ditampilkan pada:
- Laporan Data Koperasi (Print/PDF)
- Header dokumen resmi
- Sertifikat dan dokumen lainnya

## Catatan

Jika logo tidak tersedia, sistem akan otomatis menyembunyikan area logo tersebut sehingga tidak mengganggu tampilan dokumen.
